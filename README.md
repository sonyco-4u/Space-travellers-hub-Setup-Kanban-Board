# 📖 [space-travellers-hub-setup-kanban-board] <a name="about-project"></a>

> Describe your project in 1 or 2 sentences.

**[space-travellers-hub-setup-kanban-board]** The Space Traveller's hub is web application that works with real live data from the SpaceX API. The app is for a company that provides commercial and scientific space travel services. The application allows users to book rockets and join selected space missions.
